## Description

This project makes use of the [Nest.js](https://nestjs.com/) framework.
A progressive Node.js framework for building efficient, reliable and scalable server-side applications.
And used [Nest.js TypeScript starter](https://github.com/nestjs/typescript-starter) as a boiler plate.

Requires MYSQL to be installed and server to be running.

## Installation

```bash
$ npm install
```

## Running the app

Configure the .env file

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
